# 跳空统计分析工具

一个基于 Streamlit 的轻量级 Web 工具，用来统计指定时间范围内的跳空信号，并按统一交易规则回测止损、止盈和时间退出。

当前版本同时支持两种数据来源：

- `Excel / CSV`：适合大多数直接用表格存行情数据的用户
- `SQLite`：适合已经整理好数据库的用户

当前回测模式为：

- 单账户、单持仓
- 一笔买入必须对应一笔卖出
- 卖出之前不会再次买入
- 净值按已平仓交易逐笔累计

## 快慢线过滤说明

快慢线的作用是“开单过滤”，不是跳空信号本身，也不是卖出条件。

- 向上跳空时：只有当信号当天开盘价同时高于快线和慢线，才允许开单。
- 向下跳空时：只有当信号当天开盘价同时低于快线和慢线，才允许开单。
- 如果不满足对应条件，该跳空信号直接过滤，不进入交易。
- 为了避免使用未来数据，快线和慢线都使用“信号日前已经形成的收盘均线”，不使用信号当天收盘价。

## 运行方式

```powershell
pip install -r requirements.txt
streamlit run app.py
```

## 文件或数据库字段要求

程序会优先自动识别以下常见字段名：

- 日期：`date` / `trade_date`
- 股票代码：`stock_code` / `ts_code` / `symbol` / `code`
- 开盘价：`open`
- 最高价：`high`
- 最低价：`low`
- 收盘价：`close`
- 成交量：`volume` / `vol`（可选）

如果字段名不是标准命名，可以在页面里手动填写字段映射。

## 推荐数据格式规范

无论是 Excel、CSV 还是 SQLite，建议都按下面的结构整理：

| 列名建议 | 是否必填 | 说明 | 示例 |
| --- | --- | --- | --- |
| 交易日期 | 必填 | 一行代表一只股票在一个交易日的数据 | `2026-03-13` |
| 股票代码 | 必填 | 股票唯一标识 | `000001.SZ` |
| 开盘价 | 必填 | 当天开盘价 | `10.52` |
| 最高价 | 必填 | 当天最高价 | `10.88` |
| 最低价 | 必填 | 当天最低价 | `10.31` |
| 收盘价 | 必填 | 当天收盘价 | `10.66` |
| 成交量 | 选填 | 不填也可以分析 | `1256300` |

整理数据时请注意：

- 表头放在第 1 行，不要在上方再写标题行。
- 一行只能表示“一只股票 + 一个交易日”的数据。
- 不要把多天价格横向铺开成很多列。
- 不要使用合并单元格。
- 日期列推荐使用 `2026-03-13`、`20260313` 或 Excel 日期单元格。
- 如果列名和推荐格式不同，可以使用页面中的“字段映射”。

系统目前还会尽量自动识别常见中文列名，例如：

- 日期：`交易日期`、`交易日`
- 股票代码：`股票代码`、`证券代码`、`代码`
- 开盘价：`开盘`、`开盘价`
- 最高价：`最高`、`最高价`
- 最低价：`最低`、`最低价`
- 收盘价：`收盘`、`收盘价`
- 成交量：`成交量`

## Excel / CSV 使用说明

- 推荐直接上传 `.xlsx`、`.xlsm` 或 `.csv` 文件。
- Excel 文件可选填写工作表名称；留空时默认使用第一个工作表。
- 如果日期列是 Excel 常见日期格式、字符串日期、`YYYYMMDD` 或常见时间戳，程序会尽量自动识别。
- 页面中提供“下载 Excel 模板”按钮，可直接按模板整理数据。

## SQLite 使用说明

- 如果数据库里有多个行情表，可以在页面里手动填写表名；留空时会自动挑选第一个可识别的行情表。

## 当前规则说明

- 买入价固定为信号当天开盘价。
- 从买入后的第 1 个交易日开始，按“先止损、再止盈、最后时间退出”的顺序逐日检查。
- 如果未来不足 `N` 个交易日，该信号直接剔除。
- 如果到第 `N` 个交易日收盘时，收益仍不低于“到期最低目标涨幅”，则这笔信号不会生成完整交易结果，当前版本会直接剔除。
- 页面里提供了可选增强模式：如果希望第 `N` 天无论是否达标都结束交易，可以切换为“第 N 天按收盘价结束交易”。
- 如果同一时间段内出现多个信号，系统会按时间顺序只执行一笔交易；持仓期间出现的其他信号会被跳过。
- 净值曲线目前采用“已平仓净值”口径：平仓日更新净值，持仓期间净值保持不变。

## 模块说明

- `data_loader.py`：读取 SQLite 行情数据
- `rules.py`：跳空筛选和单笔交易模拟
- `analyzer.py`：遍历股票并生成明细与汇总
- `exporter.py`：导出 Excel
- `app.py`：Streamlit 页面
